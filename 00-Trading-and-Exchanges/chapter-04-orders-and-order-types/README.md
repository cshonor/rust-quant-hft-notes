# Ch 4 交易指令与订单类型 · Orders

> **Trading and Exchanges** · Larry Harris · **精读**

**指令 (Orders)** 是交易者无法亲自协商时，向经纪人或交易所传达的 **交易意愿与条件**——构建一切策略的积木。理解指令 = 理解 **流动性从哪来、代价是什么**。

**核心权衡：** **执行速度（索取流动性）** vs **执行价格（提供流动性）**
