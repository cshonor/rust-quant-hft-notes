# Ch 2 交易故事 · Trading Stories

> **Trading and Exchanges** · Larry Harris · **精读**

本章通过七个日常交易案例，展示股票、债券、期货、外汇市场中各类参与者 **如何实际安排并完成交易**。制度细节密集，目的是让读者直观感受交易中要解决的实际问题，为全书微观结构分析奠基。

> **读 go-dex 撮合引擎？** 先从 **[§1 散户股票交易](./notes/section-1-1-散户股票交易.md#与-go-dex理解订单簿的最好切入点)** 入手 — 限价/市价、进簿、做市商消化散户流，对应 `Order` / `Orderbook` 与 Ch 1 §4.2「限价单被吃」。**机构差异（暗池、冰山、FIX、拆单）** → [§2 · 机构 vs 散户](./notes/section-2-2-机构股票交易.md#散户-vs-机构核心差异不在订单类型)。
