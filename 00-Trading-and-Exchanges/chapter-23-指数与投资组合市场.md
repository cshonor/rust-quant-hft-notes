# Ch 23 指数与投资组合市场 · Index and Portfolio Markets

> **Trading and Exchanges** · Larry Harris · **选读** · Part VII

本章探讨 **指数交易** 为何成为 20 世纪最重要的金融创新之一，以及指数产品如何改变 **流动性、投资策略与价格发现**。

> **HFT 读者：** **ETF arb、index futures lead、program trading、rebalance flow** 的理论背景；与 [Ch 17 套利](./chapter-17-套利者.md)、[Ch 15 sunshine](./chapter-15-大宗交易者.md)、[Ch 22 被动 vs 主动](./chapter-22-绩效评估与预测.md)、[Ch 14 指数低 adverse selection](./chapter-14-买卖价差.md) 衔接。

## 1. 价格指数的构建 (Price Indexes)

价格指数表征 **一组工具的平均价格表现**。

### 1.1 两种常见类型

| 类型 | 英文 | 权重逻辑 | 典型例子 |
|------|------|----------|----------|
| **价格加权** | Price-weighted | 与 **成分股价格总和** 成正比 — **高价股影响最大** | DJIA、日经 225 |
| **市值加权** | Value / Cap-weighted | 与 **总市值** 成正比 — **大市值影响最大** | S&P 500 |

### 1.2 恒定除数 (Constant Index Divisor)

更换成分股或 **股票拆分**（价格加权尤需）时，用 **恒定除数** 调整，避免指数 **不自然跳空**。

| HFT 视角 |
|----------|
| **Index rebalance / reconstitution** — 成分变更 → **可预测 utilitarian flow** |
| **Divisor corporate actions** — 数据管道须正确处理 |

## 2. 为什么选择指数化投资？(The Argument for Indexation)

用 [Ch 8](./chapter-08-为什么人们要交易.md) **零和博弈** 解释被动投资的吸引力。

### 2.1 主动管理的必然劣势

| 特征 | 主动型经理 |
|------|------------|
| **目标** | 聪明交易 **击败市场** |
| **换手** | 常 **>100%/年** |
| **管理费** | 通常 **1%–3%** |
| **零和结论** | 扣除佣金与费用后，主动经理 **整体平均** 必然 **低于** 市场指数 |

→ [Ch 22](./chapter-22-绩效评估与预测.md) · [Ch 21 交易成本](./chapter-21-流动性与交易成本衡量.md)

### 2.2 被动管理的优势

| 特征 | 被动型（指数基金） |
|------|-------------------|
| **策略** | **买入并持有**，极少交易 |
| **换手** | 通常 **<10%/年** |
| **费用** | **<0.15%** |

### 2.3 残酷的现实

指数基金因 **股息再投资、申赎摩擦** 会 **略跑输** 目标指数，但因 **规避高昂交易成本**，仍常 **击败约 3/4 主动经理**。

| HFT 视角 |
|----------|
| **Passive AUM 增长** → **predictable rebalance flow**、**lower single-stock toxicity** 在指数层 |
| 主动 **alpha** 须在 **费用 + 执行成本** 之上 — 门槛极高 |

## 3. 指数市场的极高流动性

指数市场（如 **股指期货**）能以 **极低成本** 提供 **巨大流动性**：

### 3.1 知情交易者风险极低

| 原因 | 很少有人掌握 **整个市场的内幕宏观信息** |
|------|----------------------------------------|
| **做市商** | **逆向选择风险极小** → 不必 **宽价差** 自保 → **极窄 spread** |

→ [Ch 14 信息不对称](./chapter-14-买卖价差.md) — 指数 = **低 info asymmetry**

### 3.2 双向订单流匹配

| 机制 | 集中 **全市场** 买卖需求 |
|------|--------------------------|
| **效果** | 交易活跃 → 做市商 **快速周转库存** → **库存风险低** |

→ [Ch 19 双向 flow](./chapter-19-流动性.md)

### 3.3 交易便捷

**单笔** 指数产品 vs **逐只** 成分股 — **更便宜、更高效**。

| HFT 视角 |
|----------|
| **ES/NQ** 等 — HFT MM **主战场**之一 |
| **Basket trading** 成本驱动 **ETF premium/discount** 边界 |

## 4. 价格发现与程序化交易

### 4.1 衍生品引导现货

| 观察 | 指数衍生品 **流动性极高**、聚焦 **宏观风险** |
|------|---------------------------------------------|
| **结果** | 发现指数风险价格 **快于** 底层个股 |
| **传导** | 价格变化常 **先** 出现在衍生品 → **再** 传导现货 |

| HFT 视角 |
|----------|
| **Futures lead cash** — **index arb**、**lead-lag** 策略 |
| **Co-lo on futures** 有时优于 **逐股** feed |

### 4.2 程序化交易 (Program Trading)

在现货市场 **组装 / 拆解** 庞大指数组合时：

| 定义（典型） | **≥15 只股票** 且 **价值 >100 万美元** 的 **一篮子同步** 计算机下单 |
|--------------|---------------------------------------------------------------------|

→ [Ch 17 套利](./chapter-17-套利者.md) · index arb / ETF creation

| HFT 视角 |
|----------|
| **Portfolio trade / basket algo** — 与 **single-stock HFT** 不同的 infra |
| **Correlation spike** 日 — program **同步冲击** 成分股 |

## 5. 指数产品的三大形式 (Index Products)

| 形式 | 特点 | 缺点 / 注意 |
|------|------|-------------|
| **开放式共同基金 (Open-end mutual funds)** | 与基金 **直接** 申赎；按 **日终 NAV** | **无法日内** 交易；须 **现金缓冲** 应对资金流 |
| **ETF** | 交易所 **连续交易**；机构可 **creation/redemption** 一篮子股票 | 日内灵活；**税务** 常优于开放式 |
| **衍生品 (期货 / 期权)** | **现金结算** 指数期货、期权 | **保证金** 杠杆；长期须 **展期 (Roll)** 成本 |

### 5.1 ETF 机制（HFT 要点）

```
二级市场交易 ETF 份额
    ↔  NAV / IOPV
    ↔  Creation/Redemption（一篮子成分股）
        ↓
套利者收紧 premium/discount — Ch 17
```

| HFT 视角 |
|----------|
| **ETF arb desk** — futures + basket + ETF 三腿 |
| **Authorized Participant (AP)** flow — **utilitarian**、部分 **sunshine**（Ch 15） |

## 6. 与全书概念对照

| 概念 | 本章体现 |
|------|----------|
| 零和 + 费用 | 被动击败多数主动 |
| 低 adverse selection | 指数 **窄 spread** |
| 套利 / porter | ETF、index futures **收敛** |
| Utilitarian flow | 指数 **再平衡、申赎** |
| 价格发现 | **衍生品 lead** |
| Program / block | **一篮子** 执行 |

## 7. 本章总结

| 要点 | 含义 |
|------|------|
| **指数类型** | 价格加权 vs 市值加权；**恒定除数** |
| **指数化逻辑** | 零和 + 高换手费用 → 被动 **结构性优势** |
| **指数流动性** | 低宏观 informed risk + 双向 flow + **单笔便捷** |
| **价格发现** | 衍生品常 **领先** 现货 |
| **Program trading** | 大规模 **同步篮子** |
| **三产品** | 开放式基金 · **ETF** · 期货/期权 |

> **HFT 读者 takeaway：** **日历上标出 index rebalance、ETF creation 窗口** — 多为 [Ch 15](./chapter-15-大宗交易者.md) **sunshine utilitarian flow**。**Futures–cash basis** 是 [Ch 17](./chapter-17-套利者.md) 纯套利经典。**单股 HFT** 与 **指数层 HFT** 是不同 feed、不同 adverse selection 模型。
