# 1. bpftrace 是什么

| 对比 | **BCC** | **bpftrace** |
|------|---------|--------------|
| **典型形态** | Python + 嵌入 BPF C、70+ 预置工具 | **一门脚本语言** + CLI |
| **上手成本** | 读 man、理解工具参数 | 单行即可开测 |
| **适合** | 生产 runbook、复杂多探针工具 | **ad hoc**、假设验证、教学演示 |
| **本书** | [Ch 4](../../chapter-04-bcc/) | **本章** |

**全书分工：** Ch 3 清单 → Ch 4 BCC 工具箱 → **本章语言** → Ch 6+ 按资源域（CPU/内存/网络…）展开 **具体场景与工具**。
