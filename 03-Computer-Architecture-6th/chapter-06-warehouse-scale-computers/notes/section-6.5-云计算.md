## 6.5 云计算：公用计算的回归

### 规模经济

**AWS 等** 依托 WSC：

| 优势 | 量级直觉 |
|------|----------|
| 存储、带宽、运维摊销 | 比小规模 DC 便宜约 **5–7×** |
| 多租户 | 提高利用率 |

**公用计算回归：** 分时共享大型设施，客户按实例付费。

---

### 虚拟机 (VM) 的作用

| 功能 | 说明 |
|------|------|
| **租户隔离** | Xen/KVM 等 |
| **资源计量** | vCPU、内存配额 |
| **硬件抽象** | 底层换代对客户透明 |
| **产品线** | 多档位实例（计算优化、内存优化…） |

客户将 **过度配置风险** 转移给云商；云商用 **统计复用** 获利。

| HFT 视角 |
|----------|
| **研究/回测/CI** — 云合适；**低延迟实盘** — 裸金属 colo，避免 hypervisor 与多租户噪声 |
| VM **尾延迟** 与 **CPU steal** — 与 WSC 长尾同源（→ 6.2） |
| 云行情数据服务 = WSC 产品；**本地 FPGA/NIC 行情** = 反 WSC 思路 |
