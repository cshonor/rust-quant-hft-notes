## 7.7 Google Pixel Visual Core (IPU)

### 定位

| 属性 | 说明 |
|------|------|
| **设备** | **PMD** — 手机成像管线 |
| **类型** | **图像处理单元 (IPU)** |
| **计算** | **2D PE 阵列** + 模板/卷积类运算 |

---

### 关键结构

| 机制 | 作用 |
|------|------|
| **2D Line Buffers** | 滑动窗口行缓冲 — 卷积/stencil **局部复用** |
| **Halo（晕圈）通信** | 边界像素在 PE 间交换 |
| **2D SIMD** | 匹配图像空间并行 |

**原则体现：** 专用片上缓冲 + 最简 2D 并行 + 窄类型。

| HFT 视角 |
|----------|
| PMD 级 DSA — 与 colo 服务器无关 |
| **Line buffer / stencil** 思想可类比 **滑动窗口指标**（VWAP、rolling vol）— 软件实现时注意 **复用 vs 重算** |
| SOC 集成（→ 7.8）：多 IP 块共存 — 类比 **CPU + NIC FPGA + GPU** 异构机 |
