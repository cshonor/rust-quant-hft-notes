## 7.6 Intel Crest (训练加速器)

### 定位

| 属性 | 说明 |
|------|------|
| **目标** | **DNN 训练**（非推理尾延迟） |
| **数值格式** | **Flexpoint** — 16 位矩阵 + **共享 5 位指数** |
| **内存** | **HBM2** 高带宽 |
| **互连** | 极高 **片间带宽** — 多芯片扩展 |

**Flexpoint 直觉：** 比 FP32 省位宽，比纯 INT 动态范围好 — 训练数值稳定性与密度折中。

**目标：** 训练速度相对 CPU **两个数量级** 提升。

| HFT 视角 |
|----------|
| **研究侧**：大规模因子/深度学习训练 — 与实盘路径分离 |
| HBM + 多芯片 = **带宽 bound 训练**；实盘 tick 仍是 **延迟 bound** |
| Flexpoint 思路：领域定制 **数值格式** — 交易定点（fixed-point）在 embedded 风控里偶有类似 |
