## 7.3 典型领域：深度神经网络 (DNN)

### 核心运算

| 运算 | 说明 |
|------|------|
| **矩阵/向量乘** | 占算力主体 |
| **激活函数** | ReLU 等非线性 |
| **卷积** | 可转化为矩阵乘（im2col 等） |

---

### 三类网络

| 类型 | 结构 | 典型用途 |
|------|------|----------|
| **MLP** | 全连接 | 基础特征映射 |
| **CNN** | 卷积 + 池化 | 计算机视觉；**模板计算 (Stencil)** |
| **RNN/LSTM** | 循环状态 | 语音、序列、时间依赖 |

---

### 训练 vs 推理

| 阶段 | 特点 |
|------|------|
| **Training** | 反向传播、大批量、FP32/混合精度、极高算力与 HBM 带宽 |
| **Inference** | 延迟敏感、可 **量化**、批大小常更小 |

**优化手段：**

| 技术 | 作用 |
|------|------|
| **Batching** | 提高利用率 |
| **Quantization** | FP32 → 定点/INT8，推理加速 |

| HFT 视角 |
|----------|
| **alpha 研究**：离线训练 CNN/RNN/Transformer — GPU/Crest 类 |
| **实盘推理**：若用 ML 信号，多为 **低 QPS、严格尾延迟** — 更像 TPU **推理** 问题，非训练 |
| 订单簿/LOB 多为 **规则逻辑**；ML 多在 **特征/信号层**，与 DNN 硬件选型解耦 |
| Stencil 直觉可类比 **二维 order book 快照** 局部窗口 — 但 HFT 少用 CNN 于 tick 路径 |
