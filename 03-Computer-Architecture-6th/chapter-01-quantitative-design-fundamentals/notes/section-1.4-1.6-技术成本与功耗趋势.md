## 1.4–1.6 技术、成本与功耗趋势

### 1.4 技术趋势

集成电路、DRAM、Flash、磁盘、网络等技术 **不同步演进**。

**重要经验法则：**

> **带宽（吞吐量）的增长速度 ≥ 延迟（响应时间）改善速度的平方**

| 含义 | HFT 直觉 |
|------|----------|
| 链路「标称 Gbps」涨得快 | 单次事务延迟难同比下降 |
| 内存容量涨、随机访问延迟难降 | **局部性** 比「买更大 DRAM」更关键 |
| 10G→100G 网卡 | 仍要盯 **P99 尾延迟**、中断/coalescing，不能只看线速 |

---

### 1.5 功耗与能量

功耗与能量已成为微处理器设计的 **首要制约**。

| 概念 | 说明 |
|------|------|
| **动态功耗** | 与开关活动、电压、频率相关；\(P \propto C V^2 f\) 量级直觉 |
| **DVFS** | 动态电压频率调节 — 降频降压换能效 |
| **暗硅 (dark silicon)** | 散热限制下，芯片上许多晶体管 **无法同时全速开启** |

| HFT 视角 |
|----------|
| **Turbo / 睿频** 受功耗墙限制 — 多核全负载时单核峰值频率可能下降 |
| `isolcpus` + 绑核：减少无关核唤醒，降低 **package 功耗竞争** |
| 机房 **TDP / 散热** 决定能否长期维持峰值频率 — colo 选型要看 sustained vs burst |

---

### 1.6 成本趋势

硬件成本受 **时间、产量、商品化** 影响：

| 因素 | 作用 |
|------|------|
| **学习曲线** | 产量翻倍，单位成本下降 |
| **良率** | 缺陷率影响可用 die 数 |
| **商品化** | 标准品（DRAM、SSD）比定制 ASIC 便宜 |

**芯片成本核心公式（直觉）：**

\[
\text{芯片成本} \propto \frac{\text{晶圆成本}}{\text{良率} \times \text{每片晶圆合格 die 数}}
\]

面积越大 → 每片晶圆 die 数越少 → 单颗成本越高。

| HFT 视角 |
|----------|
| 实盘用 **成熟服务器平台**（商品化）通常优于早期定制硅 — 除非延迟收益可量化覆盖 NRE |
| FPGA/智能网卡：**性能/瓦特/成本** 三角要用 **TCA 式思维** 算账（类比交易 TCA） |
