## 1.1–1.2 引言与计算机分类

### 技术趋势的转变

| 背景 | 含义 |
|------|------|
| **Dennard 缩放定律终结** | 晶体管缩小不再等比例降低功耗密度 |
| **摩尔定律放缓** | 晶体管密度增长变慢，单靠缩小工艺换性能越来越难 |
| **ILP 红利见顶** | 超标量、乱序、深流水线带来的指令级并行收益递减 |

**行业转向：** 不再指望「免费」提频与 ILP，转而挖掘：

| 并行维度 | 缩写 | 典型手段 |
|----------|------|----------|
| 数据级并行 | **DLP** | SIMD、向量、GPU |
| 线程级并行 | **TLP** | 多核、多线程 |
| 请求级并行 | **RLP** | 多请求流水线、WSC 规模扩展 |
| 领域特定 | **DSA** | 为特定 workload 定制硬件（如 AI 加速器、FPGA 行情解析） |

| HFT 视角 |
|----------|
| 单机 HFT：**TLP（绑核、少共享）+ 数据局部性** 比再挖 ILP 更现实 |
| 行情解析、checksum、批量解码 → **SIMD/DLP** 候选 |
| 交易所侧 colo：**服务器（Server）** 类别；策略研究机：**桌面/PMD** |

---

### 五大计算类别

| 类别 | 典型场景 | 设计侧重 |
|------|----------|----------|
| **物联网 / 嵌入式** | 传感器、MCU | 功耗、成本、实时性 |
| **个人移动设备 (PMD)** | 手机、平板 | 能效、电池 |
| **桌面计算** | PC、工作站 | 性价比、交互响应 |
| **服务器** | 数据库、Web、**交易** | 吞吐、可靠性、扩展 |
| **集群 / 仓储级 (WSC)** | 超大规模数据中心 | 规模经济、每瓦性能 |

**HFT 实盘机** 落在 **服务器**；回测/研究集群可能触及 **WSC** 思路（批量任务、成本/瓦特优化）。

---

### Flynn 分类法

按 **指令流 (I)** 与 **数据流 (D)** 划分：

| 类型 | 指令 | 数据 | 例子 |
|------|------|------|------|
| **SISD** | 单 | 单 | 传统单核顺序执行 |
| **SIMD** | 单 | 多 | AVX/NEON 向量指令 |
| **MISD** | 多 | 单 | 少见（流水线不同阶段对同一数据） |
| **MIMD** | 多 | 多 | 多核多线程 |

| HFT 视角 |
|----------|
| 多策略进程 / 多核绑核 → **MIMD** |
| 固定宽度行情解码、批量价格运算 → **SIMD** 优化候选 |
| 热路径常是 **SISD 精心调优** + 外围 **MIMD** 分工（I/O 核 vs 策略核） |
