## ⑤ 字节序 · Byte Order

| 序 | 最低地址存什么 |
|----|----------------|
| **大端 BE** | **最高有效字节** |
| **小端 LE** | **最低有效字节** |

| 架构 | x86 = **小端** · 许多其他架构 = **大端** |

| 规则 | **绝不** 在代码里假设本机序 |

| 宏（示例） | 作用 |
|------------|------|
| **`cpu_to_be32(x)`** | CPU 序 → **大端** |
| **`le32_to_cpu(x)`** | **小端** → CPU 序 |

```c
hdr->len = cpu_to_be16(len);      /* 线上大端 */
val = be32_to_cpu(hdr->magic);
```

→ 用户态：`htons`/`ntohl`（同一数学）
