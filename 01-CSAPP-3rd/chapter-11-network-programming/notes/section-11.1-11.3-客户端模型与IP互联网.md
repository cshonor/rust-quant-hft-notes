## 11.1–11.3 客户端模型与 IP 互联网

### 11.1 客户端-服务器编程模型

```
客户端进程  ←——网络——→  服务器进程
  发起连接              监听、accept、服务
```

- **套接字 (socket)** — 连接的端点，本质是 **fd**（→ [Ch 10](../../chapter-10-system-io/)）
- HFT：**行情客户端** 连交易所 feed；**订单网关** 作 TCP 客户端连券商

### 11.2 网络

- **LAN / WAN** — 以太网、交换机、路由器
- **协议分层** — 应用 / 传输 / 网络 / 链路（细节 → [10-TCP-IP](../../../10-TCP-IP-Illustrated-Vol1/)）

### 11.3 全球 IP 互联网

#### 11.3.1 IP 地址

- **IPv4** — `struct in_addr`，32 位，常写 **点分十进制**
- **网络字节序** — **大端**；用 `htonl/ntohl`、`htons/ntohs`（→ [Ch 2](../../chapter-02-representing-information/)）

#### 11.3.2 互联网域名

- **DNS** — 域名 → IP；`getaddrinfo` 现代做法（11.4.7）

#### 11.3.3 互联网连接

| 协议 | 特点 | HFT |
|------|------|-----|
| **TCP** | 可靠、有序、字节流 | 订单、管理通道 |
| **UDP** | 无连接、报文边界 | 组播行情、低延迟 tick |

- **端口** — 16 位，区分同机多服务；`IP:port` 四元组标识 TCP 连接
