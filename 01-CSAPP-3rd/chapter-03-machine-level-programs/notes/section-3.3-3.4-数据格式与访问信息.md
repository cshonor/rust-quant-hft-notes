## 3.3–3.4 数据格式与访问信息

### 3.3 数据格式（x86-64）

| C 类型 | 汇编后缀 | 寄存器片段（64 位名） |
|--------|----------|----------------------|
| `char` | b | `%al` … |
| `short` | w | `%ax` … |
| `int` | l | `%eax` … |
| `long` / 指针 | q | `%rax`, `%rdi` … |

- **指针 8 字节** — 地址与整数用同一套寄存器传递

### 3.4.1 操作数指示符

| 类型 | 示例 | 含义 |
|------|------|------|
| 立即数 | `$0x4` | 常数 |
| 寄存器 | `%rax` | 寄存器值 |
| 绝对地址 | `0x601000` | 常数地址（少见） |
| 间接 | `(%rax)` | 内存 `*rax` |
| 基址+偏移 | `8(%rbp)` | `rbp+8` |
| 变址 | `(%rax,%rcx,4)` | `rax + rcx*4` |

**数组访问：** `A[i]` 常编译为 `lea` / 变址寻址 — 热循环里可见。

### 3.4.2–3.4.3 数据传送指令

- `mov` — 源→目的（不能 mem→mem）
- `movz` / `movs` — 零扩展 / 符号扩展加载较小类型
- `lea` — **只算地址、不写内存**（也常用于 `x + k*scale` 快速算术）

### 3.4.4 压入和弹出栈数据

- **栈向低地址增长** — `%rsp` 指向栈顶
- `pushq %rax`：`rsp -= 8`，写入 `(%rsp)`
- `popq %rax`：读出，`rsp += 8`
- `call` = push 返回地址 + jump；`ret` = pop 到 PC

**HFT：** 深调用栈、大栈帧 → cache miss、页 fault；热路径 **内联、尾调用、少 alloca**。
