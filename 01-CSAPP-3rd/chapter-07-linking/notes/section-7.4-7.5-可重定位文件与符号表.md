## 7.4–7.5 可重定位文件与符号表

### 7.4 可重定位目标文件

ELF `.o` 典型节 (sections)：

| 节 | 内容 |
|----|------|
| `.text` | 已编译机器码 |
| `.data` | 已初始化全局/静态变量 |
| `.bss` | 未初始化全局（仅占位，不占文件字节） |
| `.symtab` | 符号表 |
| `.rel.text` / `.rel.data` | **重定位条目** |
| `.debug_*` | 调试信息（`-g`） |

- **节 (section)** — 链接器视角；**段 (segment)** — 加载器视角（`readelf -l`）

### 7.5 符号和符号表

| 符号类型 | 例子 |
|----------|------|
| **全局函数/变量** | `main`, `int global` |
| **外部引用** | `extern` 声明引用的符号 |
| **局部** | `static` 函数/变量 — 其他文件不可见 |
| **节名** | `.text`, `.data`（链接器内部） |

```bash
nm main.o
objdump -t main.o
```

- **C++ name mangling** — `nm` 看到 `_Z...`；用 `c++filt` 解码

**HFT：** 链接错误常是 **符号未定义 / 重复定义**；`nm -u` 查 undefined。
