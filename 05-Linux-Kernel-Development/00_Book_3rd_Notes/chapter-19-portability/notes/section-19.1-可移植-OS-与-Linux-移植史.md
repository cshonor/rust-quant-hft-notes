## ① 可移植 OS 与 Linux 移植史

| 路线 | 说明 |
|------|------|
| **折中** | 核心接口尽量 **架构无关 C** |
| 性能路径 | **`arch/`** 下 **高度优化** 甚至 **汇编** |

| 历史 | 始于 **i386** → **Alpha、MIPS、SPARC、ARM、PowerPC…**（20+ 架构） |

```
通用内核代码（mm/ fs/ net/…）
        │
        ▼
arch/arm64/  arch/x86/  …  — 启动、页表、原子、屏障实现
```

→ **Ch 2** `arch/` 目录 · **Ch 12** per-arch 页大小

**HFT 对照：** 用户态协议解码（行情二进制）同样 **显式字节序、固定宽度类型** — 与内核规则同构。
