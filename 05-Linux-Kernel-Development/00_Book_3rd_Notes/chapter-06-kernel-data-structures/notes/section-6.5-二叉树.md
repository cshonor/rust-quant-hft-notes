## ④ 二叉树 · 红黑树 · `rbtree`

**红黑树（Red-Black Tree）** — 自平衡二叉查找树。

#### 性质（概念）

通过 **颜色 + 路径规则** 保持 **半平衡** →

| 操作 | 复杂度 |
|------|--------|
| 插入 / 删除 / 搜索 | **O(log n)** |

#### 内核实现的特殊性

| 事实 | 含义 |
|------|------|
| C **无泛型** | **没有** 通用「插入任意类型」函数 |
| 开发者 | 用内核 **rb_* 辅助函数** **手写** 比较/插入/删除 |
| 收益 | 按场景 **内联比较、少间接层** — 可优化热路径 |

```c
/* 概念：每个数据结构自己包一层 rb_node */
struct sched_entity {
    struct rb_node run_node;
    /* ... */
};
```
