## ④ 引发 Bug 与打印信息

| 宏/函数 | 行为 |
|---------|------|
| **`BUG()` / `BUG_ON(cond)`** | 条件真 → **故意 Oops** · 栈回溯 · 终止当前操作 |
| **`panic()`** | **更致命** — 打印后 **挂起整机** |
| **`dump_stack()`** | 只 **打印栈** — **不杀进程、不 panic** — 日常路径跟踪 |

```c
BUG_ON(ptr == NULL);           /* 绝不应发生 */
if (debug) dump_stack();       /* 我在哪？ */
```

| 选用 | |
|------|--|
| 开发断言 | `BUG_ON` |
| 生产可恢复 | `WARN_ON`（书中相关）/ 错误码返回 |
| 跟踪流 | `dump_stack` |
