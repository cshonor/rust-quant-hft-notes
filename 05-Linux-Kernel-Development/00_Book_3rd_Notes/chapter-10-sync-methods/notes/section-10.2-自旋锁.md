## ② 自旋锁 · Spin Locks

内核 **最常见** 的锁。

| 属性 | 说明 |
|------|------|
| 持有者 | **至多一个** 执行线程 |
| 争用 | **忙等（自旋）** 直到锁可用 |

#### 适用场景

| 适合 | 不适合 |
|------|--------|
| **持有时间极短** | 长时间持锁（浪费 CPU 自旋） |
| **中断上下文**（不能睡眠） | 需要睡眠等锁 |

#### 不可递归

| 规则 | 后果 |
|------|------|
| **同一 spinlock 不能二次 acquire** | **自死锁**（Ch 9） |

#### 与中断配合

ISR 里用自旋锁 → 须 **先关本地中断**，否则：

```
进程持锁 ──► 被中断 ──► ISR 再抢同一把锁 ──► 死锁
```

| API | 作用 |
|-----|------|
| **`spin_lock_irqsave()`** | 关中断 + 加锁；`flags` 保存状态 |
| **`spin_unlock_irqrestore()`** | 解锁 + 恢复中断 |

→ **Ch 7–8** `local_irq_save` · **Ch 8** 常与 `local_bh_disable` 组合
