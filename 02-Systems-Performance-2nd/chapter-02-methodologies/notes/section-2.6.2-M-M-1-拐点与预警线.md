## 2.6.2 · M/M/1 — 单线程报单与 70% 预警线

> **M/M/1** = 随机到达 + **随机服务** + **1 节点**。单线程网关、服务时间有波动时用此模型。  
> ← [概览与记号](./section-2.6.1-排队论概览与Kendall记号.md) · → [M/D/1 · M/M/c](./section-2.6.3-M-D-1-M-M-c-M-G-1.md) · [计算器](./section-2.6.4-排队论计算器.md)

---

### 核心公式

```
ρ = λ / μ
L = ρ / (1 - ρ)
W = 1 / (μ - λ)
W_q = ρ / (μ - λ)
W / W₀ = 1 / (1 - ρ)     （W₀ = 1/μ）
```

---

### 为什么拐点在 ~70%？— 单向车道

| ρ | 现象 |
|---|------|
| **~60%** | 偶尔慢车，几乎不堵 |
| **~70%** | 一辆慢 → 排队；随机波动 **快速拉长队尾** |
| **→90%** | 任何扰动 → 长队 |

**HFT：** λ burst + 处理耗时随机差 → ρ≈70% 时 P99 可从 **μs 跳到 ms** → timeout/reject。

```
等待倍数 W/W₀ = 1/(1-ρ)
  3× │         ╭────    ← ρ≈0.7 拐点
  2× │    ╭────
  1× │────
     └────────────── ρ
       0.6  0.7  0.8  0.9
```

| ρ | L | W/W₀ | HFT |
|---|---|------|-----|
| 60% | 1.5 | 2.5× | 开始吃 tail |
| **70%** | 2.33 | **~3.3×** | **拐点** |
| 80% | 4.0 | 5× | 危险 |
| 90% | 9.0 | 10× | 不可接受 |

**限流示例：** μ=10k/s，SLO 要求 W/W₀<3 → ρ<0.67 → **λ_max ≈ 6,700/s**

---

### 读法确认：到 70% 就别再往上顶

**对。** **70% = M/M/1 安全红线（预警线）** — 不是还能再榨 30%：

| 水位 | P99（示意） | 后果 |
|------|-------------|------|
| ~50% | ~5 μs | 正常 |
| **≥70%** | **~50 μs** | **大量拒单 — 不可接受** |
| 更高 | μs→ms | **排队雪崩** |

**部署留 10%~15% 缓冲：**

```
拐点（预警线）    ≈ 70%
Grafana 告警      ≈ 60%     ← 脉冲报单不易踩拐点
限流硬顶（可选）  ≈ 65~67%
```

核心：**在陡增前限流或扩容**，别等 P99 已从 5 μs 跳到 50 μs。
