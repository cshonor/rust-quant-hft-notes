## x86 ↔ ARM64 对照要点

| 主题 | x86-64（已学） | ARM64（本章） |
|------|----------------|---------------|
| 特权级 | Ring 0–3 | **EL0–EL3** |
| 系统调用 | `syscall` | **`svc`** |
| 页表 | 4/5 级 | **4 级**（类似） |
| 原子/屏障 | `lock` / `mfence` | **`ldxr`/`stxr` · DMB/DSB** |
