## 2.2 存储器技术与优化

### SRAM — 缓存的物理基础

| 特点 | 说明 |
|------|------|
| **静态** | 不需刷新，速度快 |
| **成本高、密度低** | 每 bit 需更多晶体管 |
| **用途** | 片上 **L1/L2/L3 cache** |

---

### DRAM — 主存

| 演进 | 要点 |
|------|------|
| 异步 DRAM | 早期，接口简单、带宽有限 |
| **SDRAM** | 同步时钟，突发传输 |
| **DDR** | 双倍数据速率；DDR4/DDR5 持续提升带宽 |
| **多 Bank** | 交错访问提高有效带宽 |
| **降电压** | 带宽↑的同时控制功耗 |

**趋势：** 容量与带宽增长快于 **随机访问延迟** 的下降 — 局部性愈发重要。

| HFT 视角 |
|----------|
| 多通道 DDR、正确 **NUMA 本地节点** 分配 — 带宽够但跨节点访问延迟高 |
| 行情大 buffer：**顺序读** 友好；随机指针追逐毁 DRAM 页模式 |

---

### HBM — 高带宽内存

| 特点 | 说明 |
|------|------|
| **3D 堆叠** | 多 DRAM die 堆叠，与处理器 **同封装** |
| **宽接口** | 显著 **降低延迟、提升带宽** |
| **代价** | 成本高、容量 per stack 有限 |

可作为 **巨大 L4** 或加速器紧耦合内存（见 2.3 第 10 项）。

| HFT 视角 |
|----------|
| FPGA/智能网卡、部分 AI 加速卡用 HBM — 交易主机仍以 **DDR + L3** 为主流 |
| 理解 HBM = 「带宽换面积/成本」的极端形态 |

---

### Flash 与 PCM

| 技术 | 特点 |
|------|------|
| **Flash** | 非易失；**按块擦写**、有 **磨损** 限制；已是 PMD/PC 主存（SSD） |
| **PCM** | 相变存储，有望介于 DRAM 与 Flash 之间（持久、密度） |

| HFT 视角 |
|----------|
| 日志/回放 trace 放 **NVMe SSD** — 注意写放大与 fsync 延迟，不进 tick 热路径 |
| 关键状态 **内存态**；持久化异步或专用线程 |
