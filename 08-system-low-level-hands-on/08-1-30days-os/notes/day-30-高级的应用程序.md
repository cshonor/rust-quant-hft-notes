### ① 命令行计算器 · calc.hrb

**实用工具** — 证明 Shell + API + 标准库 **能干活**。

| 能力 | 说明 |
|------|------|
| **四则运算** | + − × ÷ |
| **括号优先级** | 表达式解析 |
| **位运算** | AND / OR / XOR / 移位 |
| **进制** | 自动识别 **十进制 / 0x 十六进制** |
| **输出** | 结果 **同时十进制 + 十六进制** |

**体积：** **~1688B** — Day 27 **apilib 按需链接** + Day 29 **tek** 加持。

Day 28 **`type` / type.hrb`** — **长文不能往回滚**。

| 特性 | 说明 |
|------|------|
| **独立窗口** | 新 sheet 显示全文 |
| **方向键滚动** | 上下左右 |
| **可调** | **滚动速度** · **Tab 宽度** |
| **启动参数** | 自定义 **窗口宽 / 行数** |

**EDX 26 命令行参数** + Day 23 **窗口/键盘 API** 组合 — **mini less/more**。

### ⑤ IPL 极限改良 · 光盘启动探讨

#### 多扇区连续读

| 之前 | 之后 |
|------|------|
| **逐扇区** INT 0x13 | **一次读多连续扇区** |
| 外置软驱等 | 真机 **~10×** 读速（原书） |

→ [Day 3 IPL 读 10 柱面](./day-03-32位模式与导入C语言.md)

#### 内核/IPL 瘦身

| 手段 | 效果 |
|------|------|
| **弃用庞大 sprintf**（启动路径） | 代码变小 |
| **tek 压缩启动相关** | 载入数据更少 |
| **柱面数** | **10 → 9** — 启动 **更快** |

#### CD-ROM 启动（展望）

简要讨论 **从光盘引导** — 软盘 **1.44MB** 之外的 **下一步硬件载体**（原书收尾思考题）。
