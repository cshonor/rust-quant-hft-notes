### ① 窗口层级 · 输入焦点

多 app 多窗口 → 必须管 **Z 序** 与 **键盘给谁**。

#### 键盘

| 操作 | 行为 |
|------|------|
| **F11** | **最底层窗口** 提到 **最前**（Z 置顶） |
| **Tab** | 在窗口间 **轮转输入焦点** |
| **`key_win`** | 记录 **当前接收键入的窗口句柄** |

→ 延续 Day 17 **Tab/key_to** — 从 Console vs 主窗 → **多 GUI 窗**

1. **sheet_up** — 调到 **最顶层**

### ③ 定时器 API · noodle.hrb（EDX 16–19）

Day 12–13 定时器 **仅内核/Console** — 本日 **开放给 app**。

| EDX | 功能 |
|-----|------|
| **16** | **alloc** — 取一个 app 可用定时器 |
| **17** | **init** — 初始化 |
| **18** | **set** — 设定倒计时/超时 |
| **19** | **free** — 释放 |

**`noodle.hrb`：**

- **3 分钟** 倒计时
- 窗口 **动态显示 时:分:秒**
- 到点 **自动退出**

**演示：** app 也能 **sleep 等事件** — 不靠 busy loop。

→ [Day 12–13 TIMER 链表/FIFO](./day-12-定时器1.md)

### ④ 应用结束 · 自动清理定时器

#### Bug

**noodle** 设了 timer → 用户 **提前 × 关 app** → timer 仍 fire → 超时数据进 **Console FIFO** → **乱码**。

#### 修复

**定时器结构加「属主 task」标记** — app **正常退出 / 强杀 / × 关闭** 时：

```
遍历该 task 的所有 timer → cancel/free
```

**资源生命周期 = task 生命周期** — 与 Day 23 **sheet 清理** 同一原则。

**HFT：** **进程 exit 关 fd、join 线程、取消 pending alarm** — 防 **use-after-free / 幽灵回调**。
