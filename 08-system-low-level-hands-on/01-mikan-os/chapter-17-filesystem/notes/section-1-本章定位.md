### 一、从 Ch16 CLI 到持久化

| Ch 16 | **Ch 17** |
|-------|-----------|
| **echo / clear / lspci** — 内存与 PCI | **`ls`** — **读引导介质上的文件** |

### 二、本章讲什么

| 主题 | 要点 |
|------|------|
| **文件系统本质** | 大字节流 → **文件** |
| **FAT** | 与 **UEFI / BIOS** 兼容 · 公开 · 易测 |
| **BPB + Directory Entry** | 定位 **根目录簇** · 解析 **8.3 名** |
| **Block I/O 预读** | 引导器把 **卷镜像** 搬进内存 |
| **`ls`** | 终端列出根目录 |
