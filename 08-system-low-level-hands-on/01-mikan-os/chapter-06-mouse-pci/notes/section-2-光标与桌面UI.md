## 2. 光标与桌面 UI

---

### 一、鼠标光标形状

用 **二维字符数组** 定义点阵（再映射为像素）：

```
'@' → 黑色（边缘）
'.' → 白色（内部）
' ' → 透明 / 不画
```

| 步骤 | 说明 |
|------|------|
| **定义模板** | 如 12×19 光标 bitmap |
| **DrawCursor(x,y)** | 遍历模板 → 调 **PixelWriter** 上色 |
| **与 Ch4–5 复用** | 同一帧缓冲绘图 API |

---

### 二、桌面与任务栏

让系统 **「像 OS」** — 不仅是纯色背景：

| 函数 | 用途 |
|------|------|
| **`FillRectangle(x,y,w,h,color)`** | **填充** — 蓝色桌面、任务栏色块 |
| **`DrawRectangle(...)`** | **空心框** — 边框装饰 |

```
┌──────────────────────────── 桌面（蓝） ────┐
│  [光标]                                    │
│                                            │
├──────────────────────────── 任务栏 ────────┤
└────────────────────────────────────────────┘
```

→ [Ch4 PixelWriter](../chapter-04-pixel-make/notes/section-4-PixelWriter与vtable.md)

---

### 三、与后续擦除逻辑的关系

移动光标时需要 **用背景色覆盖旧位置** — 若背景是 **复杂矩形/任务栏**，简单「填背景色」会 **画错或破坏底图**（见 [6. 遗留问题](./section-6-轮询输入与遗留问题.md)）。

**本章暂用：** 记住光标下 **像素颜色** 或假定纯色桌面 — **权宜之计**。
