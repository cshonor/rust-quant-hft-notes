# Day 7 · FIFO 与鼠标控制


> **原书第七章** · **短 ISR** + **FIFO 环形缓冲** + **鼠标激活** — 键码进缓冲、鼠标原始字节「瀑布」打印。

---

### 本节三段结构

| 段 | 做什么 | 带走什么 |
|----|--------|----------|
| **① 键码 + 快中断** | 端口 **`0x0060`** | **1E/9E** · ISR **只收不发** |
| **② FIFO** | 排队 → **环形缓冲** | **`FIFO8`** 通用结构 |
| **③ 鼠标** | 键盘控制器 **`0xd4`/`0xf4`** | **IRQ12** · **128B FIFO** |

## 本日小结

| 问题 | 答案 |
|------|------|
| 键码从哪读？ | 端口 **`0x0060`**（如 **1E / 9E**） |
| ISR 该干啥？ | **只接收进 FIFO**，显示放 **HariMain** |
| 为何 FIFO？ | 防 **快输入覆盖**、键鼠 **并发** |
| 环形 vs 移位？ | **读写指针绕回**，不搬数据 |
| 通用封装？ | **`FIFO8`** |
| 鼠标怎么开？ | 经 **键盘控制器** 命令 → **`0xFA`** |
| 鼠标 FIFO 多大？ | **128 字节**（3 字节包 burst） |
| 今天验收？ | **鼠标动 → hex 瀑布**；指针 Day 8 |
