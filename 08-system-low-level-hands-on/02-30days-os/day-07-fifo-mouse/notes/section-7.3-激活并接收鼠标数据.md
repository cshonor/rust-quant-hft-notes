## ③ 激活并接收鼠标数据

#### 鼠标挂在键盘控制器上

历史原因：**鼠标电路附属于键盘控制器** — 激活要走 **键盘控制器端口** 发命令：

| 步骤（示意） | 含义 |
|--------------|------|
| 发 **`0xd4`** 等 | 告诉控制器「接下来给鼠标」 |
| 发 **`0xf4`** 等 | **启用鼠标数据报告** |
| 收到 **`0xFA`** | 鼠标 **ACK，激活成功** |

#### IRQ12 + 大 FIFO

| 属性 | 说明 |
|------|------|
| **IRQ12** | 鼠标硬件中断（Day 6 已映射 **INT 0x2c**） |
| **数据形态** | 一动鼠标 → **连续 3 字节/包**，量大 |
| **缓冲** | 专用 **`FIFO8`，128 字节** — 防 burst 覆盖 |

ISR：**只把 3 字节流拆进 FIFO**；主循环：**批量读出** → 屏幕 **十六进制瀑布**。

#### Day 7 里程碑

**晃动鼠标** → 屏上 **刷鼠标原始 hex** — **信号已进系统**；**指针还没跟手移动**（Day 8 解析 3 字节包 + 改坐标）。
