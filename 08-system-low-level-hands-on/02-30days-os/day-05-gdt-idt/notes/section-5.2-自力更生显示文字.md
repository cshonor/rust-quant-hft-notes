## ② 自力更生显示文字 · 告别 BIOS

32 位保护模式下 **不能再 `INT` 调 16 位 BIOS** 打字符 → **自己画字**。

#### 8×16 点阵

| 要素 | 说明 |
|------|------|
| **字形** | 每个字符 **8 像素宽 × 16 像素高**  bitmap |
| **绘制** | 对每个为 1 的位 → 往 VRAM **写对应色号** |
| **字体数据** | 外部导入 **ASCII 等** 点阵表 |

#### 字符串与格式化输出

- 编写 **整串输出** 函数（逐字符调用点阵绘制）
- 引入 C **`sprintf`** → 支持 **`%d` `%x`** 等

```c
char s[40];
sprintf(s, "mouse=%d", value);
putstr(s, x, y, color);
```

**在无调试器 OS 里，屏幕 print 变量 = 第一调试器。**

→ 对照 [01-CSAPP](../../../../01-CSAPP-3rd/) · 格式化 I/O 底层仍是 **写内存**；HFT 里类似 **日志 ring + 固定格式**。
