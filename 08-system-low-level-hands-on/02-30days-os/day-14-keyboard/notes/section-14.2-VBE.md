## ② VBE · 提高画面分辨率

Day 3–11 长期 **320×200×8**（模式 13h 一类）— 画面 **偏「复古」**。

#### VBE（VESA BIOS Extension）

| 项 | 说明 |
|----|------|
| **标准** | 显卡 **统一 BIOS 扩展接口** |
| **调用** | 实模式 **`INT 0x10`**，如 **`AX = 0x4F02`** 设模式 |
| **目标** | **640×480** 等 **高分辨率** |
| **VRAM** | 模式不同 → **基址/步长** 可能变 — 需 **`BOOTINFO` 更新** |

#### 兼容性 · 自动降级

```
尝试 VBE 640×480
    │
    ├─ 成功 → 高分辨率启动
    └─ 不支持 / 失败 → 回退 320×200（旧路径仍可用）
```

**在 `asmhead.asm` 完成** — 仍须在 **进 32 位前** 调 BIOS（Day 3/8 约束）。
