# Day 10 · 叠加处理


> **原书第十章** · **图层引擎** — 4KB 对齐 MEMMAN、**SHEET/SHTCTL**、局部刷新、循环边界倒推 → 鼠标不再撕任务栏。

---

### 本节四段结构

| 段 | 做什么 | 带走什么 |
|----|--------|----------|
| **① MEMMAN 续** | **4KB（0x1000）** 对齐 alloc/free | **AND 舍入** 替代除法 |
| **② 图层** | **SHEET / SHTCTL** | 背景→窗口→鼠标 **Z 序叠加** |
| **③ 局部刷新** | **`sheet_refreshsub`** | 只重绘 **变化矩形** |
| **④ 循环优化** | **bx0..bx1** 倒推边界 | 不遍历全图层像素 |

## 本日小结

| 问题 | 答案 |
|------|------|
| MEMMAN 新规则？ | **4KB 对齐** alloc/free · **AND 舍入** |
| 鼠标为何不再毁任务栏？ | **每层独立缓冲 + Z 序合成** |
| SHEET / SHTCTL？ | 单图层 vs **全局管理** |
| 第一刀提速？ | **局部 refresh**（~0.8% 像素） |
| 第二刀提速？ | **倒推 bx0/bx1**，循环 **不扫全表** |
| 里程碑？ | **高性能多图层引擎** → 多窗口 |
