## 4. x86-64 四级分页

---

### 一、四层结构

| 层级 | 名称 | CR3/上级索引 |
|------|------|--------------|
| **L4** | **PML4** | **CR3** 指向 PML4 物理基址 |
| **L3** | **PDP** | VA **[47:39]** 9 bit |
| **L2** | **PD** | VA **[38:30]** |
| **L1** | **PT** | VA **[29:21]** |
| **页** | **4KiB frame** | VA **[20:0]** 页内 offset |

**共 48 bit 有效虚拟地址**（x86-64  canonical 规则）。

---

### 二、硬件遍历

```
CR3 → PML4[idx4] → PDP[idx3] → PD[idx2] → PT[idx1] → PFN + offset → PA
```

| 页表项 | 关键位 |
|--------|--------|
| **Present** | 1 = 映射有效 |
| **Writable** | 可写 |
| **User/Supervisor** | 用户态能否访问 |
| **Physical PFN** | 高 bit 帧号 |

**TLB：** 最近 **VA→PA** 缓存 — 改页表后可能需 **invlpg** 或 **CR3 重载**。

---

### 三、与 Ch8 身份映射对照

| | **Ch8** | **Ch19** |
|---|---------|----------|
| **目的** | 内核 **VA=PA** 启动 | **应用 VA** 映 **任意 PA** |
| **PML4** | 通常 **单一内核表** | **+ 应用专用表** 或 **临时子树** |
| **大页** | 可选 **2MiB** 简化内核 | 应用 **4KiB** 按段精细映射 |

---

### 四、页故障（#PF）

| 原因 | 场景 |
|------|------|
| **Present=0** | 访问未映射 VA |
| **U/S 违规** | 用户态访问内核页（Ch20） |
| **Write 违规** | 写只读页 |

**本章应用区** 应在跳转前 **全部 Present + RW**（仍 **Supervisor** 直至 Ch20）。
