## 1. 本章定位

> **《从零自制操作系统》Ch 1 计算机工作原理和 Hello World** — **MikanOS 真正起点**

---

### 一、本章核心目标

| 目标 | 说明 |
|------|------|
| **最小闭环** | 无 Linux / Windows 等 OS 存在 → 屏幕显示 **Hello, world!** |
| **破除神秘感** | 证明「操作系统底层 = 机器码 + 数据结构 + 启动链」 |
| **双路径体验** | 先 **二进制编辑器** 手写 EFI → 再 **C 语言 + 工具链** 重写 |

**与 Ch 0 关系：** Ch 0 回答「个人能否做 OS」；**Ch 1 立刻动手** — 从第一天就有可运行的 UEFI 程序。

---

### 二、技术路线预览

```
手写 BOOTX64.EFI（机器语言体感）
    ↓
理解 CPU / 内存 / UEFI 启动
    ↓
C/C++ + Clang/LLD + EDK II（可维护开发）
    ↓
Ch 2 起：内存 map · 引导加载器 · 内核骨架…
```

---

### 三、与 01 川合 OS Day 1 对照

| | **01 30 天 Day 1** | **MikanOS Ch 1** |
|---|---------------------|------------------|
| 启动固件 | **BIOS**（Legacy） | **UEFI BIOS** |
| 介质 | 1.44MB **软盘映像** | **FAT U 盘** / QEMU 虚拟盘 |
| 入口文件 | 引导扇区 **512 B @ 0x7C00** | **`/EFI/BOOT/BOOTX64.EFI`** |
| CPU 模式 | 实模式 → 后期保护模式 | **x86-64 长模式**（UEFI 已建立） |
| 语言过渡 | 二进制 → **NASM 汇编** | 二进制 → **C/C++（EfiMain）** |

→ [01 Day 1 先动手操作](../../01-30days-os/day-01-boot-asm/notes/section-1.1-先动手操作.md)
