## 4. GDT 与分段

> x86-64 **长模式** 下分页为主 — **GDT 仍必需**（代码/数据段描述、TSS 等）。

---

### 一、重建 GDT

在 **OS 管理区域** 重新构建 **全局描述符表**：

| 描述符（示意） | 用途 |
|----------------|------|
| **Null** | 第 0 项 — 必须为空 |
| **Kernel Code** | 64-bit 代码段 — **CS** 指向 |
| **Kernel Data** | 数据段 — **SS/DS** 等 |
| **（后续 TSS）** | 多任务/特权切换 — 更后章节 |

**权限/粒度字段** — 长模式代码段常设为 **平坦、64-bit、可执行**。

---

### 二、`lgdt` 与段寄存器

```asm
lgdt [gdt_register]   ; 加载 GDTR（基址+限长）
; 远跳转或 lret 风格刷新 CS
; mov ss, DATA_SELECTOR
```

| 寄存器 | 作用 |
|--------|------|
| **GDTR** | **GDT 基址 + 界限** |
| **CS** | 当前 **代码段** 选择子 |
| **SS** | **栈段** 选择子 |

**与迁移配合：** **`lgdt` 必须指向已迁移、identity 映射可访问的新 GDT**。
