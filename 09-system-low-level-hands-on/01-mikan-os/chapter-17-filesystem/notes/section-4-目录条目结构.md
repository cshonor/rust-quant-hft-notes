## 4. 目录条目结构

---

### 一、目录也是文件

**FAT 目录** = 一串 **32 字节记录** 的数组（占 **一个或多个簇**）。

```
根目录簇:
[ entry0 ][ entry1 ][ … ][ 0x00 空槽 … ]
  32B      32B
```

---

### 二、Directory Entry（32 字节）

| 偏移 / 字段 | 含义 |
|-------------|------|
| **0–10** | **8.3 短文件名**（`NAME` + `.` + `EXT`） |
| **11** | **属性** — 只读 / 隐藏 / **目录** / 系统 / 卷标 … |
| **…** | 时间戳 · 大小等 |
| **首簇号** | 文件/子目录 **内容起始簇**（高/低字组合） |

**8+3 格式示例：** `HELLO   TXT` → **`HELLO.TXT`**

---

### 三、属性与过滤

| 属性位 | `ls` 常见处理 |
|--------|---------------|
| **0x00 空槽** | **跳过** — 已删除或未使用 |
| **0xE5 首字节** | 已删除文件标记 — **跳过** |
| **长文件名 (LFN)** | 属性 **0x0F** — **跳过**（本章只列短名） |
| **目录 (0x10)** | 可显示名后加 **`/`** 或标记 `[DIR]` |
| **卷标** | 通常 **不列** 或单独处理 |

---

### 四、遍历算法

```cpp
for (each 32-byte entry in root_cluster_data) {
    if (entry.name[0] == 0x00) break;      // 目录结束
    if (entry.name[0] == 0xE5) continue;   // 已删
    if (entry.attr == ATTR_LFN) continue;  // 长名项
    PrintFormattedName(entry);
}
```

**读簇：** 按 BPB 算偏移 → 从 **内存卷镜像** 拷贝 · 必要时跟 **FAT 链** 读下一簇（根目录较大时）。
