## ③ 传参技巧 · 运行速度优化

#### 问题

task B 要在 **sheet 上画计数** → 需要 **`sht_back`（图层指针）** — 由 task A 创建。

| 初版 | 改进 |
|------|------|
| 固定物理地址 **`0x0fec`** 当中转站 | **创建 TSS 时写初始栈** |

#### 利用栈「骗」CPU 当函数参数

创建 task B 的 TSS 时，预设 **初始 ESP**，在 **`[ESP+4]`** 放入 **`sht_back`** — task B **入口像被 call 带参**：

```
task_b_main(sht_back)   ← CPU 切到 B 时栈上已有参数
```

**优雅度：** 少 **魔数绝对地址** · 贴近 **正常 C 调用约定**。

#### 性能

- **减少无谓 screen refresh** → task B **计数更快**
- 作者用 **真机 benchmark** 对比（延续 Day 13–14 测速习惯）
