## ④ 加工润色 · `$` 与启动区签名

**自动填充：** 汇编里用 **`$`** 让编译器 **自动计算并填充 `0x00`**，把 **第 510–511 字节** 留给启动区魔数 **`55 AA`**（x86 **引导扇区有效标志**）。

```
软盘 / 映像布局（引导扇区，共 512 字节）
┌──────────────────────────────────────┬──────┐
│  IPL 代码 + 数据（nask 用 $ 填 0）    │55 AA │  ← 字节 510–511
└──────────────────────────────────────┴──────┘
  字节 0                              511
```

改输出文字时，只动源码里的字符串，**不必手算** 后面要填多少 `0x00` — 这是 **工具链第一次显身手**。
