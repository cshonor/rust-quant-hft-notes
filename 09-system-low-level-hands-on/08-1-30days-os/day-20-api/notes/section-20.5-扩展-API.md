## ⑤ 扩展 API · EDX 功能号路由

**一个 INT 0x40** — 多种服务 → 学 **BIOS：用寄存器带「子功能号」**。

| **EDX** | API（原书） |
|---------|-------------|
| **1** | 显示 **单字符** |
| **2** | 显示 **以 0x00 结尾** 的字符串 |
| **3** | 显示 **指定长度** 字符串 |

**汇编入口伪逻辑：**

```
INT 0x40 进入
    PUSHAD
    CMP EDX, 1 → putchar
    CMP EDX, 2 → puts0
    CMP EDX, 3 → putsn
    …
    POPAD
    返回 app
```

**EDX 32 位** → 理论上 **2³² 种功能扩展** — 后续 **开窗、读键、alloc** 都挂同一 **0x40** 即可。
