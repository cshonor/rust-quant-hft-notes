## 本日小结

| 问题 | 答案 |
|------|------|
| 本章主题？ | **读懂 Day 1 汇编** + **工程化构建** |
| 寄存器 vs 内存？ | 寄存器 **16 B 级**、**快**；内存 **大**、**慢**；`[]` 访存 |
| 显示谁做的？ | 早期靠 **`INT` → BIOS** |
| 加载地址？ | 引导扇区 → **`0x7c00`**（IBM 约定） |
| 构建怎么优化？ | **`ipl.bin`（512B）** 与整盘 **分离** + **`make`** |

**为 Day 3+ 铺路：** 寄存器/内存/中断模型清楚后，才能接 **C 语言** 写更复杂的 OS 功能。
