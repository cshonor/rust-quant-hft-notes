## ④ dir 命令 · FAT12 根目录

#### 目标

列出 **软盘映像里有哪些文件** — 为 Day 19 **加载 `.hrb` 应用** 铺路。

#### FAT12 根目录（原书布局示意）

| 项 | 值 |
|----|-----|
| **内存地址** | 加载后约 **`0x002600`** 起 |
| **条目数** | 最多 **224** 个 |
| **每项大小** | **32 字节** |

**32 字节目录项（核心字段）：**

| 偏移 | 长度 | 内容 |
|------|------|------|
| 0 | 8 | **文件名**（大写，不足 **空格** 补） |
| 8 | 3 | **扩展名** |
| … | … | **属性**（如 **`0x20`** 普通文件，**`0x01`** 只读） |
| … | 4 | **文件大小**（小端） |

```c
/* 示意 */
for (i = 0; i < 224; i++) {
    p = dir_base + i * 32;
    if (p[0] == 0 || p[0] == 0xE5) continue;  /* 空/已删 */
    /* 解析 name.ext attr size → printf */
}
```

**`dir` 输出：** 文件名 · 扩展名 · 大小 — **交互式外壳第一次读「磁盘结构」**。

→ 对照 [07-TLPI 文件 I/O](../../../../07-The-Linux-Programming-Interface/) · Linux **VFS/ext4** 是 FAT12 的远房后代
