## ① 命令行计算器 · calc.hrb

**实用工具** — 证明 Shell + API + 标准库 **能干活**。

| 能力 | 说明 |
|------|------|
| **四则运算** | + − × ÷ |
| **括号优先级** | 表达式解析 |
| **位运算** | AND / OR / XOR / 移位 |
| **进制** | 自动识别 **十进制 / 0x 十六进制** |
| **输出** | 结果 **同时十进制 + 十六进制** |

**体积：** **~1688B** — Day 27 **apilib 按需链接** + Day 29 **tek** 加持。
