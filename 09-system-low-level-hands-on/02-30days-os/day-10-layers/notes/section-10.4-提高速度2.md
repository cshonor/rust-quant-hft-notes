## ④ 提高速度（2）· 倒推循环边界

局部刷新后仍 **for 全图层宽高 + if 跳过** → **空转 CPU**。

**倒推：** 由 dirty rect 与 sheet 交集， **直接算** 要写的 **`x0..x1`, `y0..y1`（原书如 `bx0..bx1`）** — **`for` 只跑这个范围**。

| 之前 | 之后 |
|------|------|
| 遍历 64000 次，大多 `if` 跳过 | 循环次数 ≈ **脏区像素数** |
| 分支预测 miss、浪费 | **窗口/鼠标「嗖嗖」动** |

```c
/* 示意：循环边界 = 交集，而非 sheet 全尺寸 */
for (y = by0; y < by1; y++)
    for (x = bx0; x < bx1; x++)
        /* 合成像素 */
```

**两层优化叠加：**  
① **只 refresh 脏矩形**  
② **脏矩形内只 loop 必要坐标**

→ [03 SysPerf 少做无用功](../../../../03-Systems-Performance-2nd/chapter-02-methodologies/) · 量化 **减少遍历域** = 降延迟
