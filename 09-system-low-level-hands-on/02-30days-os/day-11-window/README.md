# Day 11 · 制作窗口


> **原书第十一章** · **现代窗口 UI** + **map 消闪烁** — 边界安全 refresh、**make_window8**、高速计数器、**像素→图层号** 地图。

---

### 本节四段结构

| 段 | 做什么 | 带走什么 |
|----|--------|----------|
| **① 边界 bug** | **`sheet_refreshsub` 裁切** | 鼠标出屏 **不乱画** |
| **② 窗口 UI** | **`SHEET`→`SHTCTL` 指针** · **`make_window8`** | 立体边框 · 标题栏 · **×** |
| **③ 计数器** | 去 **HLT** 满速刷新 | **只刷变化层及以上** — 背景不闪 |
| **④ map** | **像素归属表** | 下层刷新 **跳过上层像素** — **彻底无闪** |

## 本日小结

| 问题 | 答案 |
|------|------|
| 出屏错乱？ | **`sheet_refreshsub` 与屏幕矩形求交** |
| 窗口怎么画？ | **`make_window8`** — 灰阶立体 + 标题 + × |
| 背景为何闪？ | 从 **底图** 重合成 → 改为 **变化层及以上** |
| 鼠标为何还闪？ | 计数器 refresh **短暂盖住** 鼠标 |
| map 干嘛？ | **每像素记录归属层** — 写前 **跳过上层像素** |
| 里程碑？ | **无闪烁多图层窗口** + 高速动态内容 |
