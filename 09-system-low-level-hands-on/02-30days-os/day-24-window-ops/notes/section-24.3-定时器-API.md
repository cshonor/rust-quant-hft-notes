## ③ 定时器 API · noodle.hrb（EDX 16–19）

Day 12–13 定时器 **仅内核/Console** — 本日 **开放给 app**。

| EDX | 功能 |
|-----|------|
| **08** | **alloc** — 取一个 app 可用定时器 |
| **17** | **init** — 初始化 |
| **18** | **set** — 设定倒计时/超时 |
| **19** | **free** — 释放 |

**`noodle.hrb`：**

- **3 分钟** 倒计时
- 窗口 **动态显示 时:分:秒**
- 到点 **自动退出**

**演示：** app 也能 **sleep 等事件** — 不靠 busy loop。
