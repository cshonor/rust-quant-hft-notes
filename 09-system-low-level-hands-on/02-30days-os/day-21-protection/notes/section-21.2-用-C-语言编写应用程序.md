## ② 用 C 语言编写应用程序

#### 汇编 `_api_*` 包装

C 不能直接 **`INT 0x40`**（或不便）→ **`api.asm` / `api.asm`**：

```nasm
_api_putchar:    ; C 调用名
    MOV EDX, 1
    ; 参数从栈/寄存器按约定传入
    INT 0x40
    RET
```

#### C 程序入口 · 6 字节引导

**.hrb 文件布局：** C 编译后 **入口不在文件头** — 需 **boot 桩**：

| 桩代码（示意） | 作用 |
|----------------|------|
| **`CALL 0x1b`** | 跳到 **真实 C 入口** |
| **`RETF`** | 结束时 **回到 OS**（配合 OS **far-CALL**） |

**开发者写 C + 链一小段 asm** — 不必手写整 app。
